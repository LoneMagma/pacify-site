BurnLab

Latest
Stable Release V1

The Sovereign Development Environment

A complete air-gapped & AI-powered development laboratory that runs entirely from a USB drive. BurnLab transforms any PC into a self contained coding environment with local LLMs, offline documentation, and professional tooling, no internet, no installation, no data left behind.

Local Intelligence

Integrated Ollama runtime with Qwen2.5-Coder for offline code generation and debugging

Universal Reference

15GB+ of offline knowledge: Wikipedia, Stack Overflow, MDN, Arch Wiki

Zero-Trace

Runs entirely from RAM and USB. No data touches the host computer

Key Features

  • One-Command Bootstrap: Entire environment deployed via single bash script!
  • Adaptive Storage: Smart compression allows full functionality on 16GB drives
  • Intuitive Aliases: ask "How do I reverse a list?" queries local LLM with offline docs
  • Layered Architecture: BunsenLabs base --> Btrfs persistence --> Kiwix knowledge --> Quantized LLMs
BunsenLabs Linux Ollama Qwen2.5-Coder Kiwix VS Code Btrfs Python

Built this because I wanted a development environment that works anywhere! School's Computer Lab, libraries or random computers that too without leaving traces or depending on internet. The entire system fits on a USB drive smaller than your thumb.
